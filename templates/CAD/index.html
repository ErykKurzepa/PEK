Lang Inno Grip template for Siemens NX


Tired of repeating the same designing process over and over again?

With Siemens NX Product Template Studio you can configure complex products into simple and reusable product templates. The software allows to define custom dialog boxes to control part models, add advanced design controls to a template, build requirement checks, and create visual rules.

You can use Lang Inno Grip clamping system template to easily modify clamping jaws type, shape, position and diameter. Just open template file in NX and choose " Edit reusable component" to see custom dialog box that will allow you to modify assembly design. See video below for more details.
